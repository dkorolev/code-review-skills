# Contributing

This is just the skills repo — a source of standalone, self-contained AI-Assisted-Coding reviewer skills, one per directory under `.skills/`, meant to be installed into other repositories with `scsh installskills`. It carries no product code of its own.

- **If you are a human:** see [`README.md`](README.md) for what this repository is, the reviewers it carries, and how they are installed and used.

- **If you are an agent:** the one and only contributing guideline is [`PRINCIPLES.md`](PRINCIPLES.md) — the canonical specification every reviewer skill in this family must follow. Read it before you add or edit a skill, and hold your change to every rule it states.
